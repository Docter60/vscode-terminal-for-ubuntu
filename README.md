## Requirements

### Install Ubuntu WSL
1. Open the Microsoft Store App
2. Search for "Ubuntu"
3. Get "Ubuntu" (NOT the ones with version numbers!)
4. Wait for the installation to finish
5. Go through Ubuntu first time setup to create the standard UNIX username and password
